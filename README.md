# Word Games

以下の勉強目的で作成しています。

- Go 言語
- 連結リスト

## 必要な環境について

Go または Docker をご用意ください。

## 実行方法

### Go を使用する場合

```shell script
cd /path/to/word_games
go run cmd/アプリ名/main.go
```

### Docker を使用する場合

```shell script
cd /path/to/word_games
./build/run_app.sh アプリ名
```

## 開発する方へ

コードを書き始める前に `/doc` 配下の資料をお読みください。
